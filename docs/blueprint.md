## Core Features:

- Hero Section with Video: Engaging hero section with a background video showcasing the hostel, a brief description, and a clear call to action.
- Availability Search: Sticky widget that allows users to quickly search for available beds by date, number of nights, and guests. Includes quick access chips for 'Today,' 'Tomorrow,' and 'Weekend'.
- Dynamic Pricing & Room Display: Displays available rooms with pricing, remaining beds, and a 'Reserve Now' button. Rooms are ordered by price. Implements a tool which will attempt to identify if some of these beds should be 'held' by the system temporarily to avoid overbooking.
- Interactive Map Integration: Embedded map with hostel location, directions, nearby points of interest, and a link to open in Google Maps.
- AI Chatbot for FAQs: Chatbot that resolves user queries and leads the customer closer to booking. Includes search functionality and a list of FAQs.
- Instant Messaging: Link to instant messaging support for personalized assistance.

## Style Guidelines:

- Primary color: Forest Green (#2E7D32) evoking a natural and calm feeling, fitting the 'campestre' theme.
- Background color: Warm Ivory (#FAF9F7), a desaturated tone from the primary color for a gentle backdrop.
- Accent color: Wood Brown (#8D6E63), an analogous color with less saturation that provides a contrasting tone for secondary elements.
- Font: 'Roboto' sans-serif; using various weights and sizes for headings, body text, and captions to establish clear visual hierarchy, as per Material Design 2.
- Mobile-first layout based on a 4-column grid with 16dp margins and gutters.
- Elevation: Subtle shadows (1-4dp) to create depth and distinguish elements like cards and the app bar, following Material Design 2 guidelines.
- Subtle animations for transitions, button presses, and loading states to improve user experience.
