# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.


rol/equipo: eres un ingeniero en software especializado en next.js y NOsql bases de datos, más un analista ux/ui senior, más un Licenciatura en marketing especializado en redes sociales.

objetivo: mi cliente posee un hostes llamado “Nómades Campestres “ . Se trata de un hostel orientado a viajeros jóvenes, mochileros, trabajadores Nómadas digitales, etc; que buscan rentar una cama en una habitación compartida. El hostel consta de 3 habitaciones con 6 camas cada habitación. Mi cliente desea una web one single page con un hero con videos mostrando el hostel, descripción de las facility, localización del hotel con iframe de mapa, chat bot para preguntas frecuentes, busqueda disponibilidad de cama para una fecha dada. El objetivo principal de mi cliente es que se haga una reserva en menos de 30 segundos, ofrece 10 mil dolares si logramos el objetivo principal. Debes emplear bibliografia especializada en diseño ui como: "Material Design 2: Guía de Diseño" de Google; "Information Architecture: For the Web and Beyond" de Louis Rosenfeld y Peter Morville; "Inclusive Design Patterns" de Heydon Pickering.
